Для компиляции модуля едра "Hello World!" необходимо ввести в консоль
следующее: 
~$ make
~$ sudo insmod hello.ko
~$ dmesg | tail -1

В результате последней команды будет видно содержимое: Hello World! 

~$ modinfo hello.ko

А в результате данной команды можно будет увидель лицензию, ее автора и прочую
информацию.

Чтобы извлеч модуль ядра можно ввести ~$ sudo rmmod hello.ko
