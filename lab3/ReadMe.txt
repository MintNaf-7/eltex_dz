Для компиляции модуля едра необходимо ввести в консоль
следующее: 
~$ make
~$ sudo insmod sysfs.ko

Затем переходим в каталог, который указывали в файл sysfs.c
~$ cd /sys/kernel/systest

Просмотрим что хранится внутри
~$ ls
Увидим наш test, посмотрим что внутри через cat, увидим 0.
Чтобы положить туда значние 3 вводим
~$ sudo su
# sudo echo 3 > test 

Значение - это двоичная маска для моргания 3х лампочек
1(001) 2(010) 7 (111)

Потом вызвать
~$ sudo insmod  klav.ko 

Реультатом будут мигкающие лампочки в переданном test порядке. 

Чтобы извлеч модуль ядра можно ввести ~$ sudo rmmod sysfs.ko
