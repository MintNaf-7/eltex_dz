Для компиляции модуля едра необходимо ввести в консоль
следующее: 
~$ make
~$ sudo insmod proc.ko

если будет сообщение "Отказано в доступе", ввести следующее
~$ sudo su

Затем записываем 
# echo "Hello World" > /proc/hello

И читаем
# cat /proc/hello

Результатом будет наше сообщение - Hello World

Чтобы извлеч модуль ядра можно ввести ~$ sudo rmmod proc.ko
